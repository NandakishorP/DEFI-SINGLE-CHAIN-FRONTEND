# Lending Protocol Frontend

Frontend dApp for interacting with my Lending Protocol smart contracts.

## Tech Stack
- Next.js
- React
- wagmi + ethers.js
- TailwindCSS
- (Optional: The Graph for indexing)

## Features
- Connect wallet
- Deposit collateral
- Borrow assets
- View positions (health, LTV, balances)
- Liquidation UI (only dev-access right now)

## Setup

```bash
npm install
npm run dev
